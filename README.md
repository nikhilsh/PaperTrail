# PaperTrail

PaperTrail is an iOS app for gadget and appliance buyers to capture receipts and warranty proof automatically, then retrieve them instantly when needed.

## Core promise
Never lose a receipt or warranty again.

## Product direction
- SwiftUI
- MVVM
- Combine
- OCR-assisted capture
- product-centric organization
- strong local search
- iCloud durability via CloudKit-backed sync

## Current status
Project initialized from app incubator planning.

## Immediate goal
Build a disciplined v1 that proves the core loop:
1. capture receipt/warranty proof
2. extract/store useful metadata
3. search and retrieve records later
4. preserve user trust with durable sync
