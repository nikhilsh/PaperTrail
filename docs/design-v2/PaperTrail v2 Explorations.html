About this exploration

Built on the shipped Archive design system (cream paper, gold foil, Newsreader + Plex Mono). Five tracks: Settings declutter, Warranty page, IAP that pops, Notifications & permissions, App Store listing. Each track goes safe → bolder, left to right. Mix and match — everything shares the same tokens, so any combination ships coherently. The written spec for Claude Code is in handoff/DESIGN_LANGUAGE.md.
